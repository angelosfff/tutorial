About Them:

The languages: About:
HTML Hyper Text Markup Language. It is mainly used for building websites. It is not considered a programming language, instead its a markup language. Markup language refers to annotating a document (also a coding language) that is recognizable to the text.
CSS Cascading Style Sheets. This language is used to change the way a sheet looks for the code or the development. Such as the font, and placement of each aspects. CSS works very well with HTML.
JavaScript A advanced coding language, also known as JS, that is mainly used to make things such as websites, and other forms of development.
Python Python is a very simple coding language that is very popular. It is used worldwide and is mainly for beginners. It can be used to create mini games and commands
PHP General purpose scripting language. It is one of the best languages for things such as web development. The code works very well with the C language which is also its implementation. (Fun fact: Facebook was made using PHP).
C The C programming language. This is a general purpose language that is very popular amongst the coding world. The language is very similar to C++ and C#.
GO The Go language was made by formed Google employees Robert Griesemer, Rob Pike, and, Ken Thompson.
